- [TP_1_spark_shell_et_word_count.md](TP_1_spark_shell_et_word_count.md)
- [TP_2_projet_et_pre_processings.md](TP_2_projet_et_pre_processings.md)

Il est plus que recommandé de lire ces deux fichiers.
